# 🚩 Protect the Flag

A fast-paced tower defense-style game where you must protect your flag from waves of enemies!

## 🎮 How to Play

1. **Move** - Use Arrow Keys or WASD to move your player around
2. **Shoot** - Click to shoot at enemies (aim with your cursor)
3. **Defend** - Protect the flag at the top of the screen from enemy attacks
4. **Survive** - Complete waves to increase difficulty and score points

## 📊 Game Mechanics

- **Health**: You start with 100 HP. Lose all health to game over
- **Waves**: Each wave spawns more enemies than the last
- **Score**: Earn 10 points for each enemy defeated
- **Flag**: Located at the top of the screen - don't let enemies reach it!

## 🎯 Objective

Survive as many waves as possible and achieve the highest score!

## 🚀 Features

- Dynamic wave system with increasing difficulty
- Real-time enemy and bullet collision detection
- Smooth player controls and aiming system
- Score and health tracking
- Pause/Resume functionality
- Game over screen with final stats

## 🛠️ Development

Built with:
- HTML5 Canvas for rendering
- Vanilla JavaScript for game logic
- CSS3 for styling

## 📁 Files

- `index.html` - Game HTML structure and UI
- `styles.css` - Game styling and layout
- `game.js` - Core game logic and mechanics

## 🎮 Controls

| Control | Action |
